# 💊 Pharmacie

A web-based pharmacy management application built to automate the day-to-day operations of a single-pharmacist store.

## 🔧 Tech Stack
- **Backend:** PHP
- **Database:** MySQL

## ✨ Features
- **Medicine Management** — Add and manage medicines in the inventory
- **Stock Tracking** — Monitor stock levels and keep inventory up to date
- **Billing System** — Automatically calculates purchase totals for each transaction
- **Customer Records** — Stores customer details including name and mobile number

## 🎯 Purpose
This project was built to replace manual store management with a simple digital solution, reducing errors and saving time for pharmacists handling daily operations.
